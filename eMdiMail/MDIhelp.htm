Brightness

Lightens or darkens the image.

  • Range is   -100 to 100, where -100 is darkest and 100 is lightest.
[TOP]

Captured Document Location

Path to the folder where captured single images are stored. The default folder on the computer is "\My Documents\MDI". If a path is not specified, it is created.

  • Valid characters are ASCII characters 32 through 126 (decimal). Invalid characters are /:*?"<>|.
  • The string begins with a backslash, and cannot end with a backslash except for the “\” string (root folder).

ColorMode

  • None - sets color mode to grayscale
  • Monochrome_Normal - sets color mode to black and white without interpolation
  • Monochrome_Enhanced_TIFF_Quality - sets color mode to black and white with interpolation
[TOP]

ColorMode Brightness Threshold

When ColorMode is set to monochrome, determines overall brightness of image by determining the general level at which pixels are converted from greyscale to black or white. Sets the relative level at which grayscale pixels are converted to black or white when ColorMode is set to Monochrome.
  • VeryDark - Only the lightest pixels are converted to white. Results in a generally darker image.
  • Dark
  • Normal
  • Bright
  • VeryBright - Only the darkest pixels are converted to black. Results in a generally lighter image.
[TOP]

Contrast Enhancement

Controls whether the image contrast is enhanced. Adjusts brightness and contrast automatically when used.
  • None
  • Photo
  • Black_text_on_white_background
  • White_text_on_black_background

File Name Template

Defines the default file name. The total length of the folder location, file name, and file extension string cannot exceed 200 characters (location + file name + max. 4-character file extension).

You can specify the file name to consist of one or more of the following static fields:

  • Time/Date stamp $(time), in the format YYYY-MM-DD HH_MM_SS. For example, if you set the file name to “myFile_$(time)”, the captured file name would be similar to ‘myFile_2009-02-11 16_23_26.jpg”, which is 2/11/2009 4:23:26 pm for GTM-8 Pacific US time.
  • Document number $(num), which is an automatically incremented number for every file stored. For example, if you set the file name to “myFile_$(num)”, the first two file    names would be “myFile_1.jpg” or    “myFile_2.jpg”.
  • Location $(location), which is GPS information. If a location service is not available, then the text of $(location) is “no_location”.
  • Default static field is the root file name which is “doc_$(num)”, where num is an automatically incremented number. For example, if you use the default file location and file name on a handheld computer, the file path is similar to “\My Document\MDI\doc_1.jpg”.

If you do not specify the static field with a file name, the computer generates a random number in order to set a unique file name for a captured document file. For example, if you set the file name to “myFile”, the second generated file name is similar to “myFile_143553.jpg”, where 143553 is a random number generated by the computer.

Remarks
Valid characters are ASCII character numbers 32 to 126 (decimal). Invalid characters are \/:*?"<>|

FocusCheck

Controls whether the image focus is evaluated as part of determining if there is a valid document to capture.
This setting is useful in preventing blurred document captures. The focus check is performed on the document center.

  • Disable
  • Medium
  • High
[TOP]

Image Area Capture Ratio

Controls how far away from the document page  the capture can be taken. The further away the capture is taken, the smaller the page looks and the ratio of page relative to the overall image is smaller.

  • The value is expressed as a percentage with a range of 0 to 100, where the higher the value, the larger the page appears in the image.
[TOP]

Image Lighting Correction

Controls the image lighting correction. Enable this to correct for non-uniform lighting conditions.

  • None
  • Enabled
[TOP]

Image Rotation

Rotates the image counterclockwise from its original position by the specified amount.

  • None
  • Degrees_90
  • Degrees_180
  • Degrees_270
[TOP]

Perspective Correction

Controls whether or not perspective correction is used.

  • When correction is disabled, there is no projective mapping, and the system returns the smallest rectangle image containing the area to be captured. When correction is enabled, projective mapping is active and the resolution used is automatically adjusted.
[TOP]

Noise Reduction

Filters out video noise. This setting helps reduce the noise resulting from the edge enhancement process. If set to 0, this filtering stage is disabled. The higher the setting value, the higher the filtering.

  • Range is 0 to 9
[TOP]

Output Compression

Sets the output file type of the document capture image.

  • Bitmap
  • Jpeg
  • TIFF
[TOP]

Output Compression Quality

Controls the JPEG compression level for the captured image, expressed as a percentage with a range of 0 to 100.
  • Range is 0 to 100, where 100 is highest quality and largest file size.
[TOP]

Text Enhancement

Enables text enhancement, which controls the readability of text in the image.

  • None
  • Low
  • Medium
  • High
